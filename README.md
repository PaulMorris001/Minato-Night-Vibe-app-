# Minato-Night-Vibe-app
Night vibe App

To use this app clone the repo using `git clone https://github.com/PaulMorris001/Minato-Night-Vibe-app-/`
- Then you `cd mobile`
- and run `npm install`

Then move into the server directory and run `npm install` again

Then you start the backend with `npm run dev`
And the frontend with `npm start`
